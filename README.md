# Insulin FLI App

## Overview
The Insulin FLI App is a prototype application designed to detect insulin FLI (Fluid Level Indicator) for healthcare-related use cases. This project focuses on applying software-based detection logic to a real-world medical problem.

## Project Structure
- main.py : Core application logic
- assets/ : Resources and assets used by the application

## Features
- Insulin FLI detection prototype
- Basic validation workflow
- Healthcare-focused problem approach

## Technologies Used
- Python

## Status
This is an academic prototype project developed for learning and experimentation purposes. It is not intended for clinical or production use.
